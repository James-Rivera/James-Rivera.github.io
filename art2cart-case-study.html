Art2Cart

Art2Cart is a digital art marketplace connecting artists with collectors through an intuitive, gallery-inspired platform.

Role Full Stack Developer
Timeline 4 Weeks
Platform Web Application
Tools PHP, MySQL, Bootstrap
01 / MARKETPLACE

A responsive marketplace grid optimized for scanability and quick filtering. Users can browse by style, price, and artist, with fast-loading previews for a fluid discovery experience.

02 / BACKEND

Server-side PHP with MySQL powering listings, licensing, and user sessions. Queries are designed for predictable performance, with clear data contracts across endpoints.

03 / ADMIN

An admin surface for creators and moderators to manage artworks, licenses, and orders. Clean tables, inline edits, and role-based access keep workflows tight.
